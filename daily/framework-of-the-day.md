# Framework of the Day

> A rotating spotlight system to build familiarity with all frameworks over time. Each day, one framework is surfaced in the morning briefing.

## How It Works

1. The briefing template includes a "Framework Spotlight" section
2. Each day, the next framework in the rotation is surfaced
3. The spotlight includes: framework name, one-sentence summary, and a question to consider
4. Over ~6 weeks, every framework gets a day
5. The cycle repeats, but the question changes based on what's new in your status data

## Rotation Schedule
