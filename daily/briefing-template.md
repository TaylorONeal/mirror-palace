# Morning Briefing Template

> Run this at the start of each day. Takes 3-5 minutes. The agent reads status, surfaces priorities, and flags patterns.

## Protocol

### 1. State Check (30 seconds)
- **Energy level**: 1-10
- **Sleep quality**: poor / fair / good / great
- **Primary physical state**: rested / tired / wired / foggy / pain
- **Emotional weather**: calm / anxious / excited / flat / irritable

> *Updates: `status/domains/health-fitness.md` if pattern emerges*

### 2. Priority Surface (1-2 minutes)

Review today against the three-question filter:

1. **What is the single most important thing I could do today?**
   *(Apply north-star-test: does doing this make everything else easier or unnecessary?)*

2. **Is there a decision I'm avoiding?**
   *(Check `status/decisions/DECISIONS-INDEX.md` for stale pending decisions. Apply reversibility-classification — is it actually a two-way door I'm treating as one-way?)*

3. **Who do I owe a response to?**
   *(Check `status/people/` for contact frequency gaps)*

### 3. Domain Scan (1 minute)

Quick RYG check across the 10 domains. Flag any that are:
- **Red and unaddressed** — needs attention today
- **Stale** (not updated in 30+ days) — needs a check-in, even if brief
- **Recently changed** — acknowledge progress or new information

### 4. Pattern Alert (30 seconds)

The agent checks for active failure modes:
- Task snoozed 2+ times? → Freeze/overthinking pattern
- 5+ open threads across projects? → Project sprawl
- Calendar >80% packed? → Over-commitment
- Haven't done something fun in 7+ days? → Life balance drift

If a pattern is detected, name it. Don't fix it. Naming is the intervention.
