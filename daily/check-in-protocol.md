# Check-in Protocol

> 60-second state check before starting work. This anchors your current capacity so the system can match tasks to energy.

## The Check-in

Answer these 4 questions. Don't overthink them — first instinct.

### 1. Energy (1-10)
Where are you right now?
- **1-3**: Low. Fog, heaviness, resistance. → Match with lowest-friction tasks only.
- **4-6**: Medium. Functional but not sharp. → Routine work, admin, communication.
- **7-8**: Good. Clear, focused, capable. → Deep work, creative tasks, hard decisions.
- **9-10**: Peak. Rare. → Save for the hardest, most important thing.

### 2. Body State
Pick one:
- [ ] Rested
- [ ] Tired but functional
- [ ] Wired / overstimulated
- [ ] Foggy / slow
- [ ] Physical pain or discomfort
- [ ] Restless / agitated

### 3. Emotional State
Pick one:
- [ ] Calm / grounded
- [ ] Anxious / tense
- [ ] Excited / motivated
- [ ] Flat / low
- [ ] Irritable / frustrated
- [ ] Scattered / overwhelmed

### 4. Window
How much focused time do you have before the next interruption?
- [ ] <30 min
- [ ] 30-60 min
- [ ] 1-2 hours
- [ ] 2+ hours

## What the System Does with This

Based on your check-in:

| Energy | Window | Recommended Work |
|--------|--------|-----------------|
| 1-3 | Any | Rest, or smallest possible admin task |
| 4-6 | <30 min | Quick replies, filing, scheduling |
| 4-6 | 30+ min | Routine work, light creative |
| 7-8 | <60 min | One focused deliverable |
| 7-8 | 60+ min | Deep work session |
| 9-10 | Any | The hardest thing on your list |

If emotional state is **anxious** or **overwhelmed**: apply distortion-detection before starting work. Name what's causing it.

If emotional state is **flat**: don't force it. Check if physiology-as-character pattern is active. Consider embodied-awareness micro-practice before starting.

If body state is **wired**: don't start deep work yet. 5 minutes of embodied-awareness first.

## Tracking

This check-in data feeds into `status/domains/health-fitness.md` over time. Patterns emerge: which days are consistently low? Is there a time-of-day energy curve? Does energy correlate with sleep, exercise, or social activity?
