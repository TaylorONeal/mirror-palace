# Confidence Language Docs

This folder contains the language-first confidence overlay specification and implementation plan.

## Files
- `../PRD-confidence-language-overlay.md` — product requirements document for the feature.
- `confident-language-taxonomy.md` — MECE taxonomy and routing/data-contract integration.
- `confidence-scoring-system.md` — 5-dimension scoring model and rewrite logic.
- `master-prompt-confidence-mds.md` — generation prompt for regenerating the two core specs.
- `implementation-next-steps.md` — phased plan for implementation, routing changes, testing, and doc rollout.
- `references/this-not-that-communication-tips.md` — imported communication phrasebook corpus used for rewrite patterns and detector lexicons.

## Recommended read order
1. PRD
2. Taxonomy
3. Scoring
4. Implementation next steps
5. Reference corpus (This Not That)
6. Master prompt
