Mirror Palace Framework Map

A cognitive framework toolkit. 36 frameworks across 16 categories, 8 situation routes, and 6 agent archetypes. The system never prescribes a vision of the good life. It helps users discover and pursue their own.

Surface patterns → Calm the mind → Act with integrity → Live fully

Epistemology

How we think about thinking
Concept Formation ▸
Grounds reasoning in clear definitions
Use when: Reasoning feels ungrounded, concepts are vague, noise is overwhelming signal
MIRROR Cognitive Architecture ▸
Structures persistent internal narratives for agents
Use when: Designing agent cognition, structuring persistent internal narratives
Information Compression ▸
Decides what to keep vs. discard
Use when: Context is getting long, need to decide what to keep vs discard

Decision-Making

Choosing under uncertainty
Reversibility Classification ▸
Sorts decisions into one-way and two-way doors
Use when: Any decision -- first classify it before analyzing. Many agonized-over decisions are actually reversible.
Regret Minimization ▸
The 80-year-old test
Use when: High-stakes life choices, paralysis on big moves. "At 80, will you regret not doing this?"
North Star Test ▸
Forces priority ranking
Use when: Evaluating features, actions, or priorities. What matters most? What would you sacrifice for what?
Ikigai Diagnostic ▸
Maps capability, passion, need, and livelihood
Use when: Career/purpose misalignment, feeling lost about direction. Identifies which quadrant is empty.

Behavioral Psychology

Why people do what they do
Jobs to Be Done ▸
Reveals what people actually hire a product/action to do
Use when: Designing products, understanding why people do things
Behavior Equation ▸
Models motivation x ability x prompt
Use when: Building habits, designing interventions, things aren't getting done
Habit Loop Design ▸
Cue, routine, reward architecture
Use when: Creating new habits or breaking old ones
Variable Reward Schedules ▸
Explains engagement through unpredictable rewards
Use when: Designing engagement systems, maintaining motivation
Loss Aversion ▸
Models the asymmetry of gains vs. losses
Use when: Streak design, notification language, framing choices
Identity Reinforcement ▸
Connects actions to stated identity
Use when: Connecting actions to stated values, building confidence

Cognitive Therapy

Seeing thinking clearly
Distortion Detection ▸
Names common cognitive distortions
Use when: Catastrophizing, all-or-nothing thinking, should-statements. Names them clearly without pathologizing.
Linguistic Reframing ▸
Shifts perspective through language
Use when: User is stuck, needs perspective shift, communication design
Awareness as Intervention ▸
Names the pattern as the intervention
Use when: Pattern is recurring, naming it may be enough. Sometimes seeing it clearly is all you need.

Executive Function

Getting things done when the brain won't cooperate
Executive Function Model ▸
Maps which cognitive function needs support
Use when: Understanding which cognitive function needs support
ADHD Design Rules ▸
Design principles for real humans
Use when: Designing systems for real humans with executive function variance
Time Blindness ▸
Addresses the distorted sense of time
Use when: Scheduling, estimating duration, deadline language

Continuous Learning

Learning that actually sticks
Closed-Loop Learning ▸
Observe, model, test, revise cycle
Use when: Want to learn from experience, not just accumulate information

Self-Image

Who you think you are (and how that shapes everything)
Self-Image Cybernetics ▸
Models the self-concept as a thermostat
Use when: Self-concept is limiting behavior, identity doesn't match actions
Teleological Psychology ▸
Examines hidden goals and belonging strategies
Use when: Examining hidden goals, belonging strategies, purpose narratives
Systems Over Goals ▸
Replaces endpoint-chasing with sustainable practices
Use when: Building sustainable practices instead of chasing endpoints
Rational Self-Interest ▸
Grounds self-prioritization in rational ethics
Use when: Guilt about self-prioritization, values misalignment, living by default rather than choice

Route Detail Cards

Each route includes a framework sequence, stop conditions, and contraindications.

Shame / Self-Attack / Collapse

Shame spirals self-reinforce. The user generates evidence for the shame narrative, which deepens fusion, which generates more evidence.
1Needs & Feelings Clarity
2ACT (defusion)
3Distortion Detection (optional)
Guardrails

Stop if: User becomes more activated. Childhood material surfaces they're not ready for. The shame is connected to ongoing harm.

Not this route if: The shame is appropriate guilt about genuinely harmful behavior. The user needs stabilization first.

Overthinking / Fusion / Avoidance

Overthinking masquerades as diligence. The user feels productive because they're "working on it" mentally, but no action occurs.
1ACT (defusion)
2Awareness as Intervention (optional)
3Behavior Equation (optional)
Guardrails

Stop if: ACT becomes another layer of overthinking. The stuck-ness masks a real constraint.

Not this route if: The "overthinking" is legitimate deliberation on a genuinely complex decision. The rumination has a shame quality.

Conflict / Blame / Faux-Feelings

"I feel abandoned" is not a feeling. These are interpretations encoded as feelings, and they escalate conflict because each person hears accusations, not vulnerability.
1Needs & Feelings Clarity
2Stories vs Facts (optional)
3ACT (optional)
Guardrails

Stop if: NFC is being used to tone-police legitimate anger. The user is more interested in being right.

Not this route if: The conflict involves real harm. The issue is a genuine attachment rupture.

Relationship Rupture / Attachment Activation

Attachment-activated conflict gets processed at the surface level. The couple argues about content when the real issue is connection and safety.
1Needs & Feelings Clarity
2EFT Relationship Lens
3Psychological Contracts (optional)
Guardrails

Stop if: EFT language is being used to diagnose the partner. The framework is justifying staying in a harmful dynamic.

Not this route if: Active abuse or coercion. Primarily logistical conflict. Very new relationship.

Friendship Ambiguity / Mixed Signals

Friendships have their own dynamics: unclear reciprocity, invisible contracts, effort imbalances that go unspoken.
1Needs & Feelings Clarity
2Psychological Contracts
3Stories vs Facts (optional)
Guardrails

Not this route if: The bond is deep enough for attachment dynamics. The "friendship" involves manipulation.

Career Confusion / Stuckness / Drift

Career stuckness generates two loops: endless information gathering or impulsive pivots. Both avoid the harder question of what actually matters.
1Ikigai Diagnostic
2North Star Test
3Failure Modes (optional)
4Regret Minimization (optional)
Guardrails

Not this route if: The stuckness is actually depression or burnout. The user has clear direction but is blocked by a specific constraint.

Values Conflict / Decision Paralysis

Genuinely competing values make a decision feel impossible. The tradeoffs are real, not avoidance-driven.
1Reversibility Classification
2North Star Test
3Regret Minimization
Guardrails

Not this route if: The paralysis is fear-driven. The decision involves another person and relational dynamics.

Emotional Signal Unclear (Meta-Route)

The user knows something is off but can't identify what. This is the entry point for people who can't yet name their situation.
1Needs & Feelings Clarity
2Embodied Awareness
3Awareness as Intervention
→Reroute to specific route
Guardrails

Once the feeling or need becomes clearer, move to the appropriate situation-specific route above.

The 6 Agent Archetypes

Pre-built agent personalities that apply frameworks in distinct styles.

The Mirror

Pattern Detection
Names contradictions, tracks patterns across weeks, asks the hard questions. Warm but not soft.
Key: failure-modes, distortion-detection, awareness-as-intervention, regret-minimization

The Briefer

Daily Briefing
Morning state report. "What's Working" before pattern alerts. State-responsive -- minimal on hard days.
Key: north-star-test, failure-modes

The Tracker

Progress Tracking
Reports positive trends alongside gaps. Tracks reciprocity in relationships.
Key: identity-reinforcement, systems-over-goals

The Strategist

Strategic Planning
Recommends releasing goals, not just adding them. "Who else does this matter to?"
Key: ikigai-diagnostic, regret-minimization

The Operator

Domain Execution
Handles operational tasks within a specific domain. Notes stakeholder impact.
Key: behavior-equation, executive-function-model

The Watcher

Change Detection
Monitors for significant shifts. Writes change logs for other agents.
Key: awareness-as-intervention, failure-modes
