## Column Definitions

- **Domain**: The knowledge or skill area (e.g., "programming", "psychology", "leadership", "fitness", "philosophy", "language")
- **Format**: How you're engaging with it — `book`, `course`, `practice`, `mentorship`, `community`, `self-study`, `project`, `coaching`
- **Active/Paused**: Current engagement status — `active` (currently working on it), `paused` (intentionally on hold), `abandoned` (dropped, should remove or restart), `completed` (finished, now in integration)
- **Integration Level**: 0-100 rating of how much this learning has changed actual behavior. 0 = consumed but not applied. 100 = fully integrated into daily life.

## Relevant Frameworks
- `closed-loop-learning` — Is learning being tested and refined, or just accumulated?
- `identity-reinforcement` — Learning as identity vs learning as tool
- `systems-over-goals` — Building learning systems, not chasing completion
- `structured-self-coaching` — Weekly growth practice structure
- `developmental-stages` — What stage of development is this learning supporting?
- `north-star-test` — Is this learning aligned with what matters most?

## Review Cadence
Biweekly. Flag if active items haven't been touched in 21+ days (likely should be paused). Monthly review of integration levels — consuming without integrating is a pattern to watch.
