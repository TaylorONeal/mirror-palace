## Conventions

- **Issue ID format:** 3-letter domain code + hyphen + 3-digit number (e.g., `FIN-001`, `CAR-012`)
- **Domain codes:** FIN (money-finances), CAR (career-work), HEA (health-fitness), FUN (fun-recreation), ENV (environment), COM (community), FAM (family-friends), PAR (partner-love), PGR (personal-growth-learning), SPI (spirituality)
- **RYG:** R (Red, 0-39), Y (Yellow, 40-69), G (Green, 70-100)
- **Linked Issues:** Comma-separated Issue IDs that this issue affects or is affected by
- **Last Updated:** YYYY-MM-DD format
