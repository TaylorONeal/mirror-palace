# Self-Image Cybernetics

The self-image operates like a thermostat governing behavior — performance automatically rises or falls to match self-concept. Willpower-based change fails because it fights the thermostat. Sustained change requires updating the self-image first through mental rehearsal (vivid imagination of desired performance), success tracking (accumulating evidence that contradicts negative self-image), and identity updating (rewriting self-descriptions based on evidence).

## Quick Reference
- **Best for**: Breaking persistent behavior patterns, addressing self-sabotage, designing success-visibility systems
- **Complexity**: High
