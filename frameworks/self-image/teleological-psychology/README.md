# Teleological Psychology

All behavior is purpose-driven (teleological), not caused by the past. Guiding fictions — implicit beliefs about how the world works — shape decisions and strategies. Belonging strategies developed in childhood persist into adulthood. The transformative question: "What hidden goal does this behavior serve?" Restores agency by reframing patterns from flaws to strategies that can be consciously revised.

## Quick Reference
- **Best for**: Understanding persistent behavior patterns, addressing self-sabotage, examining hidden motivations
