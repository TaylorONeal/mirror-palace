# Systems Over Goals

Build daily practices instead of chasing endpoints. Goals create binary pass/fail conditions and defer happiness; systems create daily success and compound over time. Combine mediocre skills into rare talent stacks. Manage energy as the primary metric. Connect daily practices to identity — each repetition is a vote for who you are becoming. The system is the success; outcomes are byproducts.
