Use the Needs & Feelings Clarity framework.

Apply it when:
- the user's emotional language is real but muddy
- conflict analysis is getting accusation-heavy
- the user says they don't know what they need
- a reaction needs to be translated into usable signal before action

Do this sequence:
1. Separate facts from interpretation.
2. Identify faux feelings or accusation-words if present.
3. Translate those into cleaner emotional states.
4. Identify the likely need or value clusters touched.
5. Distinguish the deeper need from the preferred tactic.
6. Suggest the cleanest next action that would serve reality rather than just discharge emotion.

Guardrails:
- Do not over-therapize.
- Do not assume every feeling maps to one neat need.
- Do not let the framework delay an obvious boundary or decision.
- If the user is flooded, simplify.

Output format:
- Facts
- Clean feelings
- Likely needs / values
- Strategy being confused for the need
- Best next move
