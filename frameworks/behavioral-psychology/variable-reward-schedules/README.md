# Variable Reward Schedules

Fixed rewards lose motivational power through habituation. Variable reinforcement sustains engagement through unpredictability. Applied schedule: 80% on first daily completion, 100% at milestones, 30% otherwise. Always layer variable rewards on top of deterministic completion signals.

## Quick Reference
- **Best for**: Sustaining long-term engagement with recurring behaviors
- **Complexity**: Medium
- **Time to apply**: 15-30 minutes to design schedule
