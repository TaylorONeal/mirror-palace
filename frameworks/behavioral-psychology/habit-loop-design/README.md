# Habit Loop Design

Every habit operates through Cue > Routine > Reward. Design each phase independently: specific environmental cues, embarrassingly small routine starts, and variable rewards that maintain engagement. Fixed rewards lose power; variable reinforcement sustains the loop.

## Quick Reference
- **Best for**: Building new habits, diagnosing habit failures, designing recurring behaviors
- **Complexity**: Medium
- **Time to apply**: 15 minutes to design, 2-3 weeks to establish
- **Pairs with**: Behavior Equation, Variable Reward Schedules, North Star Test, Jobs to Be Done, Loss Aversion, Identity Reinforcement
