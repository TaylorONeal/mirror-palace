# Identity Reinforcement

Connect actions to stated values. Occasionally surface alignment — not every time, variable like a thoughtful friend noticing. Specific, warm, brief. Never comment on misalignment. Approximately 10-20% of aligned behaviors. Prioritize hard-day alignment moments.

## Quick Reference
- **Best for**: Long-term behavior sustenance through identity, agent personality design
- **Complexity**: Medium
- **Time to apply**: Ongoing (not a one-time exercise)
- **Pairs with**: Variable Reward Schedules, Habit Loop Design, Loss Aversion, Ikigai Diagnostic, Jobs to Be Done, Behavior Equation, Concept Formation
