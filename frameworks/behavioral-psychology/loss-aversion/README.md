# Loss Aversion

Powerful but dangerous motivational lever. Losses feel 2x as strong as equivalent gains. Design rules: streaks only after 3+ days, never "you failed," deadlines as countdowns not "OVERDUE," "carried forward" not "missed." Always provide guilt-free escape hatches. The anxiety test: if users dread the system, loss aversion is miscalibrated.
