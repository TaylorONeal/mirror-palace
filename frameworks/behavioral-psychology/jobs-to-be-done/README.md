# Jobs to Be Done

Every product, habit, or system is "hired" for three simultaneous jobs: Functional (what task?), Emotional (how should I feel?), Social (how does this affect relationships?). The key insight: win on the emotional job — a solution that completes the task but makes the user feel bad is failing.

## Quick Reference
- **Best for**: Product design, understanding user motivation, diagnosing engagement drops
- **Complexity**: Medium
- **Time to apply**: 15-30 minutes per analysis
- **Pairs with**: Behavior Equation, North Star Test, Habit Loop Design, Loss Aversion, Identity Reinforcement
