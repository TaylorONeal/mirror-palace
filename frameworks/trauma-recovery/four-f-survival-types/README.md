# Four-F Survival Types

A framework for understanding four fundamental survival responses -- Fight, Flight, Freeze, and Fawn -- as chronic adaptations to developmental trauma. Maps how emergency nervous system programs become personality patterns, identifies the inner critic's role in maintaining each type, and provides a process for managing emotional flashbacks.

## Quick Reference
- **Best for**: Understanding chronic stress responses and disproportionate emotional reactions
- **Complexity**: Medium
- **Time to apply**: Initial assessment 30-45 minutes; ongoing awareness practice
- **Pairs with**: Childhood Emotional Neglect, Emotionally Immature Parents, Embodied Awareness

## Files
- **theory.md** -- Four survival types, emotional flashbacks, inner critic patterns, hybrid types, recovery direction
- **template.md** -- Self-assessment scoring for each type, flashback log, inner critic pattern identification, stress cascade mapping
- **agent-prompt.md** -- Detection signals and gentle reflection protocol for agents working with survival response patterns
