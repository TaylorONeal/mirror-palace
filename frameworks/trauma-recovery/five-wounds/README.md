# The Five Wounds

A framework mapping five core childhood wounds -- Rejection, Abandonment, Humiliation, Betrayal, and Injustice -- to their corresponding defensive masks (Withdrawal, Dependence, Masochism, Control, Rigidity). Each wound includes characteristic behavioral patterns, body tendencies, relational signatures, and a healing direction oriented toward practicing the opposite of the mask.

## Quick Reference
- **Best for**: Understanding why compulsive patterns persist despite cognitive awareness
- **Complexity**: High
- **Time to apply**: Initial assessment 30-45 minutes; deep integration is long-term
- **Pairs with**: Four-F Survival Types, Embodied Awareness, Enneagram
