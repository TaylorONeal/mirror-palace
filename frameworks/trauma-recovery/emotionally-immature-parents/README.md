# Emotionally Immature Parents

A framework mapping four types of emotionally immature parents (Emotional, Driven, Passive, Rejecting) and two primary adaptation patterns in children (Internalizer, Externalizer). Traces the development of the role-self -- the identity constructed to survive parental limitations -- and the path back to the true self through recognition, grief, experimentation, and boundary development.

## Quick Reference
- **Best for**: Understanding how parental emotional limitations shaped your patterns and identity
- **Complexity**: Medium
- **Time to apply**: Initial assessment 45-60 minutes; integration is ongoing
- **Pairs with**: Four-F Survival Types, Childhood Emotional Neglect, Family Systems Differentiation

## Files
- **theory.md** -- Four parent types, internalizer/externalizer patterns, role-self vs. true self, recovery path, dealing with EI parents as an adult
- **template.md** -- Parent type assessment, adaptation style identification, role-self mapping, healing fantasy check, recovery actions
- **agent-prompt.md** -- Detection signals and gentle pattern-recognition protocol for agents working with parental adaptation patterns
