# Personal Failure Modes

Seven generic recurring failure patterns: Freeze/Overthinking, Project Sprawl, Over-Commitment, Self-Worth Tied to Status, Emotional Editing, Physiology as Character, and Romanticizing Other Selves. Each with trigger, appearance, disguise, and intervention. Naming reduces power -- what you can see, you can choose not to follow. Template includes space for defining custom failure modes.

## Quick Reference
- **Best for**: Identifying and interrupting recurring self-defeating patterns
- **Complexity**: Low (7 modes, each with trigger-disguise-intervention structure)
- **Time to apply**: 15-20 minutes for identification; ongoing for tracking
- **Pairs with**: Structured Self-Coaching (weekly pattern tracking), Big Five (trait prediction), Enneagram (motivational roots)
