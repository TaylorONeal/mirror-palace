# Psychological Contracts

Invisible, unspoken agreements that operate beneath awareness: "I do X; therefore Y should happen." The other party never agreed to the terms. Four components: your contribution, the expected return, the enforcement mechanism (resentment, withdrawal, martyrdom), and the belief underneath. Resentment is the primary detection signal. Excavation methods include following the resentment, following the performance, following the body, and sentence completion.

## Quick Reference
- **Best for**: Identifying hidden expectations driving relational friction and disproportionate resentment
- **Complexity**: Medium (concepts are simple; honest self-examination is hard)
- **Time to apply**: 20-30 minutes for initial excavation; ongoing as contracts surface
- **Pairs with**: Enneagram (core fears generate contracts), Embodied Awareness (body detects violations), Failure Modes (contracts maintain modes)
