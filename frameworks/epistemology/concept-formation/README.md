# Concept Formation

Form clear, grounded concepts by identifying essential characteristics and omitting measurements. Build knowledge hierarchically from concretes to abstractions. Use integration (non-contradiction) as the test of truth.

## Quick Reference
- **Best for**: Resolving conceptual confusion, building definitions, grounding abstract discussions
- **Complexity**: High
- **Time to apply**: 10-30 minutes for a full worksheet; instant for essentialization habit
- **Pairs with**: Information Compression, Mirror Cognitive Architecture, Jobs to Be Done
