# Embodied Awareness

Body-based observation practices focused on awareness, not change. The body is an information source that detects, processes, and communicates data the conscious mind often misses. Five daily micro-practices (body scan, breath observation, sensation tracking, transition check, contact point awareness) build the skill of reading the body's report. The core instruction: notice without fixing.

## Quick Reference
- **Best for**: Developing the body as an information source and breaking cognitive-only processing loops
- **Complexity**: Low (simple practices, consistency is the challenge)
- **Time to apply**: 1-5 minutes per micro-practice; daily consistency
- **Pairs with**: Subconscious Repatterning, Four-F Survival Types, Structured Self-Coaching
