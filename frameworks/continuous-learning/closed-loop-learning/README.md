# Closed-Loop Learning

A four-step cycle that converts experience into behavioral change: Capture (collect signals from real work), Model (distill into testable hypotheses), Test (run small reversible experiments), Integrate (update systems with validated insights). Prefer small experiments over grand redesigns. Learning is behavioral change, not information accumulation.

## Quick Reference
- **Best for**: Turning recurring problems into experiments, building calibration data, operationalizing insight
- **Complexity**: Medium
- **Time to apply**: 15-30 minutes per cycle; weekly cadence is ideal
- **Pairs with**: Awareness as Intervention, Time Blindness, Executive Function Model, Systems Over Goals
