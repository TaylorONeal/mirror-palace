# Reversibility Classification

Classify decisions as one-way doors (irreversible, deserve analysis) or two-way doors (reversible, decide quickly). The primary failure mode is treating reversible decisions as irreversible — over-analyzing low-stakes choices while real opportunities pass.

## Quick Reference
- **Best for**: Decision triage, breaking analysis paralysis, speed calibration
- **Complexity**: Low
- **Time to apply**: 2-5 minutes
- **Pairs with**: Regret Minimization, Information Compression, Concept Formation, North Star Test
