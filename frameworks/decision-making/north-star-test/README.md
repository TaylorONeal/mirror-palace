# North Star Test

For every feature or action: Does the user immediately know what to do, feel confident, do it, and feel good? Tracked through leading indicators (time to first action, completion rate) and anti-metrics (session >15min, snooze >40%).

## Quick Reference
- **Best for**: Feature evaluation, design decisions, user experience assessment
- **Complexity**: Medium
- **Time to apply**: 10-20 minutes per feature/interaction
- **Pairs with**: Behavior Equation, Habit Loop Design, Loss Aversion, Variable Reward Schedules, Jobs to Be Done
