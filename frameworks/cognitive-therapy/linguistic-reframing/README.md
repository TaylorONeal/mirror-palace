# Linguistic Reframing

Five language patterns that shift perception from stuck states to action states: future pacing (connect action to outcome), chunking down (micro-steps to overcome initiation barriers), anchoring (state-aware task matching), pattern interrupt (gentle intervention for stuck loops), and presupposition (assume positive action in how questions are framed).

## Quick Reference
- **Best for**: Designing action prompts, notifications, and task presentations that support rather than hinder motivation
- **Complexity**: Medium
- **Time to apply**: 5-15 minutes per message or interface element
- **Pairs with**: Distortion Detection, Awareness as Intervention, ADHD Design Rules, Executive Function Model
