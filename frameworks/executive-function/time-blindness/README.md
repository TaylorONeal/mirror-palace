# Time Blindness

Interventions for impaired time perception: replace relative time with concrete, always show duration estimates, compare available time to task time, use countdowns instead of dates, schedule concretely instead of intending flexibly, and build calibration data over time. The core principle is to make time visible externally rather than relying on internal temporal sense.

## Quick Reference
- **Best for**: Scheduling, deadline management, task presentation, and any temporal communication
- **Complexity**: Low (principles are simple; consistent application is the challenge)
- **Time to apply**: Immediate for individual interactions; 30 minutes for a full system audit
- **Pairs with**: ADHD Design Rules, Executive Function Model, Distortion Detection, Closed-Loop Learning
