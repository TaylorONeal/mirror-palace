# ADHD Design Rules

Ten principles for designing systems that work with attention regulation variability rather than against it: one thing at a time, hide the backlog, external accountability, time anchoring, transition support, effort visibility, energy matching, escape hatches, variable rewards, and no shame language. Not accommodations — design corrections for how attention actually works.

## Quick Reference
- **Best for**: Auditing and building task presentation systems, notifications, and productivity interfaces
- **Complexity**: Medium
- **Time to apply**: 20-40 minutes for a full audit; individual principles can be applied immediately
- **Pairs with**: Executive Function Model, Time Blindness, Distortion Detection, Linguistic Reframing
