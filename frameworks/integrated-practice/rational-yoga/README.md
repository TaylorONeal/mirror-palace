# Rational Yoga

A complete mind-body-ethics framework that preserves classical yoga's practical techniques — breath regulation, attention training, physical practice, ethical principles, pattern observation — while grounding every concept in observable, repeatable reality. Supernatural claims are replaced with psychological and physiological explanations; the practices remain intact because they work through mechanisms that don't require belief.

## Quick Reference
- **Best for**: Building an integrated daily practice across physical, mental, and ethical dimensions — without mysticism
- **Complexity**: High (9 progressive phases, 80+ concepts)
- **Time to apply**: Ongoing practice; initial orientation takes 2-4 weeks per phase
- **Pairs with**: Embodied Awareness, Awareness as Intervention, Systems Over Goals, Identity Reinforcement
