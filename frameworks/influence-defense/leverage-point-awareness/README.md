# Leverage Point Awareness

A defensive framework for understanding leverage dynamics -- how power asymmetries in resources, information, emotional investment, options, and social position create compulsion disguised as choice. Maps the leverage cycle, distinguishes genuine generosity from strategic positioning, and provides tools for responding to leverage without escalating.

## Quick Reference
- **Best for**: Recognizing when you are being leveraged and building agency within power asymmetries
- **Complexity**: Medium
- **Time to apply**: 20-30 minutes per situation analysis
- **Pairs with**: Behavioral Signal Reading, Manipulation Watchouts, Psychological Contracts

## Files
- **theory.md** -- Leverage anatomy, the leverage cycle, recognizing leverage in real-time, help vs. control, context-specific leverage, response strategies
- **template.md** -- Asymmetry mapping, help evaluation, compliance check, response planning
- **agent-prompt.md** -- Leverage detection and options-building protocol for agents
