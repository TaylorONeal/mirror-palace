# MBTI and Cognitive Functions

A framework mapping sixteen personality types through eight cognitive functions (Se, Si, Ne, Ni, Te, Ti, Fe, Fi) arranged in a four-function stack: dominant, auxiliary, tertiary, and inferior. The inferior function -- least developed and most vulnerable -- is both the Achilles heel and the growth path. Includes function stacks for all sixteen types, inferior function grip patterns under stress, and growth direction.

## Quick Reference
- **Best for**: Understanding cognitive processing preferences, stress patterns, and growth through the inferior function
- **Complexity**: High (eight functions, sixteen types, four-position stacks)
- **Time to apply**: 30-45 minutes for type and function identification
- **Pairs with**: Big Five (trait data), Enneagram (motivational data), Four-F Survival Types (stress interaction)
