# Big Five Personality Model (OCEAN)

The most empirically validated personality framework, measuring five continuous dimensions -- Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism -- each with six facets (30 facets total). The profile, not any single dimension, describes the personality. Includes cross-framework implications for survival types, failure modes, and environment design.

## Quick Reference
- **Best for**: Granular self-understanding and environment design based on trait profile
- **Complexity**: Medium (five dimensions, thirty facets)
- **Time to apply**: 30-45 minutes for full facet-level assessment
- **Pairs with**: Enneagram (motivational depth), MBTI (cognitive functions), Failure Modes (vulnerability prediction)
